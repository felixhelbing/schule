* Schul-Arbeitsblätter

Arbeitsblätter für die Grundschule. Generiert mit [[https://typst.app][Typst]].

** Fertige PDFs

Die aktuellen PDFs stehen als Download bereit:

[[https://github.com/felixhelbing/school-worksheets/releases/latest][Latest Release (PDFs)]]

Bei jedem Push auf =main= werden die PDFs automatisch neu gebaut und als =latest=-Release veröffentlicht.
Der Workflow kann bei Bedarf auch manuell über GitHub Actions gestartet werden.

|---------------+---------------------------------------------------+---------------------------------------------------------------|

** Lokal bauen

*** Quickstart

#+begin_src sh
./build.sh
#+end_src

Die erzeugten PDFs liegen danach in =out/=.

*** Voraussetzungen

- Unix-kompatibles System (Linux, macOS, Windows WSL)
- [[https://typst.app][Typst]] >= 0.14.2

** Struktur

#+begin_example
in/
  satzbau/         → satzbau.typ
  schreibuebung/   → schreibuebung.typ
  wortarten/       → wortarten.typ
  leseuebung/      → leseuebung.typ
#+end_example

Pro Kategorie gibt es ein Typst-Template und ein Eingabeverzeichnis mit YAML-Dateien.

Gemeinsame Einstellungen (Seite, Schrift, Absatz) stehen in =common.typ=.
